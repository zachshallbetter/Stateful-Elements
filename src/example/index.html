Stateful Element Documentation

Installation

Install the package using npm:

npm install stateful-element

Usage

Import the package and use the StatefulElement class:

import { StatefulElement } from 'stateful-element';

Create a new instance:

const element = new StatefulElement();

API

  • createState(stateName, initialValue)
  • setState(stateName, newValue)
  • getState(stateName)
  • subscribe(stateName, callback)
